In this secion of my notes I will descibe some of the main tags and their uses. Tags are the code known as Markup that allows you to modify the content of your web page. There are dozens of tags that were not covered but the specific tags covered were as follows:

TAGS

BOLD - Bold is the code written in order to make the desired text in your HTML bold for emphasis. The Bold tag is written"< b >" and is closed by writing "< /b >". Bold is also an "inline" element. (which is to say that it does not create a box around the desired text.)

EMPHASIS - Emphasis is the code written in order to make the desired text in your HTML document italicized. This tag is written as "< em >" and is closed by writing "< /em >". Emphasis is also an "inline" element.

Next we went through a number of "attributes". Attributes differ from tags as style elements can be added to attributes to modify their look. There are many more than are listed below.

ATTRIBUTES

Anchor - Anchor is used to add hyperlinks into your documnents. Anchor is writen as "< a href="url" > contents < / a>". Anchor is an "inline" element.

Image - Image is used to add pictures into your documnents. Image is writen as "< img src="url" alt="text" >" and there is no closing statement. Image is an "inline" element.

Whitespace - White space in your code does not provide changes to the look of your website. Whitespace is defined as White space is the empty spaces in a design. White space is used to separate disparate design elements and group similar ones. White space is the lack of graphics or text in the layout.

Break - Break is used to drop future text down one line into your documnents. Break is writen as "< br >" and there is not closing statement. Break is an "inline" element.

Paragraph - Paragraph is used to drop future text down one line into your documnents. Paragraph is writen as "< p >" and the closing statement is "< /p >". Paragraph is a "block" element. (which is to say that a box is created around the text.)

Span/Div - Span and Div simialrly to Paragraph and Break in your documnents. Span is written "< span class="foo" >" and closed with "< /span >". Div is written "< div class="bar" >" and closed with "< /div >". The term class is used to add styles to your statements. Span is an "inline" element and Div is a block element.
